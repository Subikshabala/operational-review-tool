# Web-Based Operational Review Tool (ORT)

## Project Overview
The Operational Review Tool (ORT) is a web-based system designed to monitor, review, and improve student performance through structured feedback and performance metrics.

## Roles
- User (Student): View performance metrics and reviews
- Staff: Provide reviews and feedback
- Admin: Input and manage performance data

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js + Express (Phase 2)
- Database: MySQL (Phase 2)
